Question 4
  • Difference between == and === with example.

Answer:The == operator is capable to change the values being checked by performing some type conversions which JS thinks as human error but in case of ===, no conversions are performed and it is considered as a strricyt equality.

Like, 20=="20" will return True as RHS will be treated as number.

Whereas, 20==="20" will return False as RHS will be treated strictly as string.
