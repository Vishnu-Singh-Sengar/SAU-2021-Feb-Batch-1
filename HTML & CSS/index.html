YouTube Scraper

Creating a smart YouTube recommendation engine using YouTube Data API V3, which fetches you the N most top videos of any search query user provides. It works on a set of complex algorithms and rules through which every video has to pass through and is then assigned a score accordingly. The top N videos are then returned to the user.

Smart Air Ambulances

Creating AI backed software using GPT2 and NLP which could be embedded in Medical Drones to provide fast and effective aid to the people in need. AI will be capable to handle end to end random conversations and can tell the aid from its manually curated database consisting of various first aid methods. In the later stages, I am planning to include computer vision into it so that the drones could monitor the live health of the patient and tell the aid accordingly.

InstaDaily

InstaDaily - Hours of Scrolling.... SIMPLIFIED!! InstaDaily is your one stop software to fetch top images out of a particular hashtag on Instagram. User can also save his favorites by saying "yes" or any such similar words, after an image is displayed. Program also works on a complex architecture to ensure you wont see the same image again in your entire life. Program also has inbuilt tools with which you can clear the cache by just a click. Clearing cache will still enable the program to memorize what images you have seen earlier so that you wont see them again as promised. Start using it straight away. Sort It Out!

Developed a smart conveyor based out of IoT and OCR which sorts out the packages moving on it by there destination address. It captures the address using a camera mounted at the start of a conveyor and accordingly triggers the motor corresponding to it. The package moves along the conveyor, hits the bar attached to the motor and falls off the conveyor through a chute into a collecting box, thus sorting it out.
